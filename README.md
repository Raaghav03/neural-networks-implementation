# neural-networks-implementation
Black Box and White Box Implementation of Neural Networks for Fashion MNIST Dataset
## Black Box Model:
hidden_layers: $2 (128, 64)$
epochs: $10$
test_acc: $88\%$
test_loss: $0.3341$
<br>
## White Box Model:
hidden_layers: $1 (128)$
epochs: $10$
test_acc: $85\%$
test_loss: $0.3649$
